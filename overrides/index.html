A minimal ASGI web framework

Jackie is a framework that focusses on small easily integratable components in an asynchronous context. This means that views and/or routing written using Jackie can easily be used together with other ASGI software. It's minimal nature makes it easy to understand and mix and match with other components to create a complete web application ecosystem.
